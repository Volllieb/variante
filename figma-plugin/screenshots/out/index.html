📸 Screenshot Studio

Jede Datei ist eine standalone HTML-Page (1280×800) mit Figma-Frame-Mock. Sie zeigt die exakte Plugin-UI eingebettet in eine realistische Figma-Desktop-Umgebung.

📋 So machst du die Screenshots

  1. Öffne eine der unten verlinkten HTML-Dateien im Browser (Vollbild)
  2. Browser-DevTools: Ctrl+Shift+I → Device Toolbar Ctrl+Shift+M → 1280×800 einstellen
  3. Zoom auf 100% (wichtig — kein Browser-Zoom!)
  4. Screenshot: DevTools → "Capture screenshot" oder Win+Shift+S → Rectangular Snip
  5. Als PNG speichern, in figma-plugin/ ablegen, im Publish-Dialog hochladen

🎬 Demo-GIF (15–30s, max 5 MB)

Empfohlen: LICEcap (kostenlos) — festes 1280×800 Frame, nimmt direkt als GIF auf.

Alternative: ShareX (kostenlos) — Screen Recording → GIF export.

Per CLI: ffmpeg -f gdigrab -framerate 10 -video_size 1280x800 -i desktop -t 25 output.gif

Was zeigen: Kompletten Flow — Plugin öffnen → Test anlegen → Element picken → Variant B editieren → Snippet installieren → Dashboard/Results.

Optimierung: gifsicle -O3 --lossy=80 -o output.min.gif output.gif (unter 5 MB bleiben).
