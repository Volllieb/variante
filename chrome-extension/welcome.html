✅ Element Picker installed

This extension is part of the variante Figma plugin. You normally don't need to open it directly.

  1. Start a test in the Figma plugin. Create an A/B test there — your site opens automatically with the picker active.
  2. Click an element. Hover over the live page and click the target element. ESC cancels.
  3. Back to Figma. The tab closes itself after selection.
The extension popup (click the icon) is only the manual fallback in case the page doesn't open automatically — you can paste a testId from the plugin there.
