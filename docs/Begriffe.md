## Begriff	Was das ist





# Onboarding
  Der erste Einstieg: Du gibst deine URL ein, siehst sofort eine KI-Variante deiner Seite — noch bevor du dich anmeldest oder das Snippet einbaust. Der Aha-Moment kommt zuerst.
# Health Check	
  Deine permanente Status-Seite im Dashboard. Zeigt dir: Lebt das Snippet? Ist das Figma-Plugin verbunden? Ist deine Domain verifiziert? Kein einmaliges Gate — du kannst jederzeit zurückkommen und prüfen.
# New test
  Der Wizard zum Erstellen eines neuen A/B-Tests. 6 Schritte: Test-Details → Snippet → Element → Design → Generate → Goal. Aktuell läuft der im Figma-Plugin, später auch direkt im Dashboard.
# Dashboard
  Dein täglicher Arbeitsplatz. Hier verwaltest du Tests, checkst Results, verwaltest Billing und Account.

# Temp Session	
  Du kannst einen Test erstellen, ohne dich anzumelden. Deine Daten bleiben temporär erhalten, bis du einen Account erstellst.
